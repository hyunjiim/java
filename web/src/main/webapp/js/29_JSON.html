JSON 관련 프로퍼티와 메소드를 제공하는 클래스 함수

JSON 정의

JSON은 데이터 구조와 실제 데이터를 다른 언어 및 플랫폼에서 해석 가능한 형식으로 전송할 수 있는 데이터 교환 포멧

사람이 읽을 수 있는 문서로 이루어졌으며, 코딩도 더 짧고 처리속도가 빠른 경량 언어

임시 데이터 저장에 적합

모든 유형의 프로그래밍 언어를 위한 데이터 포맷으로 사용될 수 있어 고도의 상호 운용성 제공

데이터베이스에 저장되어 있는 고객 우편 주소의 유효성을 검증하기 위해 해당 주소 데이터를 JSON 형식으로 주소 검증 서비스 API에 전송 가능
