Mouse Event

click: 태그에서 마우스 버튼을 누른 경우 발생되는 이벤트

dblclick: 태그에서 마우스 버튼을 두번 연속으로 누른 경우 발생되는 이벤트

mousedown: 태그에서 마우스 버튼을 누르는 순간 발생되는 이벤트

mouseup: 태그에서 눌려진 마우스 버튼을 놓는 순간 발생되는 이벤트

mouseover: 태그에 마우스 커서가 진입하는 순간 발생되는 이벤트 - 버블링 단계 포함

mouseout: 태그에 진입된 마우스 커서가 벗어나는 순간 발생되는 이벤트 - 버블링 단계 포함

mouseenter: 태그에 마우스 커서가 진입하는 순간 발생되는 이벤트 - 버블링 단계 미포함

mouseleave: 태그에 진입된 마우스 커서가 벗어나는 순간 발생되는 이벤트 - 버블링 단계 미포함

hover: 태그에 마우스 커서가 진입하거나 벗어나는 순간 발생되는 이벤트

mousewheel: 태그에서 마우스 휠이 움직이는 동안 발생되는 이벤트

mousemove: 태그에서 마우스 휠이 움직이는 동안 발생되는 이벤트

contextmenu: 태그에서 마우스 오른쪽 버튼을 누르면 발생되는 이벤트
